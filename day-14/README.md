# Day 13 - Text Processing 1: grep
 
## Objective
Learn how to search for patterns inside files using grep.
 
---
 
## What I Learned
- grep searches for a word or pattern inside a file
- grep -i makes the search case insensitive
- grep -r searches recursively through folders
- grep -n shows the line number of each match
 
---
 
## What I Built / Practiced
- Used grep to search for words inside files
- Combined grep with pipes to filter command output
