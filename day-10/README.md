# Day 10 - Viewing & Editing Text: nano, vim, less

## Objective
Learn how to view and edit text files directly in the terminal.

---

## What I Learned
- nano is the beginner friendly text editor in Linux
- vim is powerful but has a steeper learning curve
- less lets you scroll through large files without loading all at once

---

## What I Built / Practiced
- Opened and edited a file with nano
- Used less to scroll through a large file

---

## Challenges Faced
- Limited time this week — full lesson to follow

---

## Key Takeaways
- nano is the go-to editor for beginners
- less is better than cat for large files
- Press q to exit both vim and less
