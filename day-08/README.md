Day 08 - Working with Files: cat, head, tail, wc, file

## Objective
Go deeper with files — reading content, inspecting parts
of files and understanding what's inside them.

---

## What I Learned
- touch creates an empty file instantly
- cat shows the full contents of a file
- head shows the top lines, tail shows the bottom lines
- wc counts lines, words and characters in a file
- file tells you what type of file something is

---

## What I Built / Practiced
- Created data.txt using echo and >>
- Read it with cat, head and tail
- Counted its contents with wc
- Used file to check the file type

## Key Takeaways
